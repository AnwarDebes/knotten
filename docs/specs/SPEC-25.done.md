# SPEC-25 completion note

## What was built

A coherent set of board-ready concept deliverables in plain Norwegian, generated from the platform's figures so every number traces back to the same sources used on the site. Built by `scripts/build-deliverables.mjs` into `public/deliverables`.

- **Energikonsept_Knotten.docx**: the technical and energy concept (passivhus, sol, lagring, deling, V2G/V2H, robusthet).
- **Mulighetsstudier_Knotten.docx**: a feasibility brief per option (solar, wind, geothermal, storage, shared base/energy sharing, V2G/V2H), each with assumptions, sources and a verification note.
- **Energimodell_Knotten.xlsx**: an editable model with live formulas (demand from area, solar from kWp and yield, self-sufficiency), PVGIS- and SSB-sourced, that the contractor can adjust as plot count and design firm up.
- **Reguleringsplan_anbefalinger_Knotten.docx**, **Robusthet_og_beredskap_Knotten.docx**, **Markedsstrategi_og_prospekt_Knotten.docx**, **Oversikt_for_Sigve_Simonsen_AS.docx** (non-technical), and **Personvernerklaering_Knotten.docx** (matching the actual consent flow and cookieless analytics).
- **Presentasjon_kommune_og_partnere_Knotten.pptx**: a minimal, self-explanatory deck.
- **Sammendrag_en_side_Knotten.pdf**: the one-pager derived from the set.
- A `manifest.json` (file, size, format) the download kit reads.

## Verification

- Local gate green: lint, type-check, format, tests (143), build, bundle budget.
- The generator runs clean and emits all ten documents plus the manifest; each file downloads over HTTP with the correct size and type (docx, xlsx, pptx, pdf verified).
- Figures mirror the site modules and research (PVGIS yield, SSB baseline, passivhus targets, NO2 schemes), so the documents, site and spreadsheet do not diverge.

## Honesty and privacy

Every document opens with a forbehold: all figures are indicative (June 2026), require professional verification, and are not an offer; plot count, prices and progress are stated once the zoning is ready. Sources are listed in each document. No personal data; no fabricated facts or imagery. The deliverables are regenerated by re-running the script if the underlying figures change.
